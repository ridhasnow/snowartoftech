# Snow Art Of Tech - متجر الألعاب الإلكترونية

## 🎮 وصف المشروع

موقع متجر احترافي متخصص في بيع الماس والرصيد للألعاب الإلكترونية. الموقع يتميز بتصميم عصري وحديث مع واجهة مستخدم سهلة وجذابة.

## ✨ المميزات

- 🎨 **تصميم احترافي وحديث**: واجهة جميلة ومتناسقة
- 📱 **Responsive Design**: يعمل بشكل مثالي على الهاتف والحاسوب
- 🖼️ **بانر صور متحرك**: تتغير الصور تلقائياً كل ثانيتين
- 🛍️ **قسم المنتجات**: عرض المنتجات بطريقة احترافية
- 🎯 **نوافذ منبثقة (Modal)**: عرض معلومات المنتج والتسعيرة
- 📰 **قسم الأخبار**: مخصص لنشر الأخبار والفعاليات
- 📅 **قسم الأحداث**: عرض الأحداث والمسابقات
- 📞 **التواصل المباشر**: روابط مباشرة للواتساب والفيسبوك ويوتيوب
- 🎯 **دعم اللغة العربية**: واجهة كاملة باللغة العربية

## 🖼️ متطلبات الصور

### أبعاد الصور الموصى بها:

- **Banner Images** (banner1.png, banner2.png, banner3.png):
  - الحجم الموصى به: **1920 × 400 بكسل**
  - الصيغة: PNG أو JPG
  - الحد الأدنى: 1200 × 300 بكسل

- **Product Images** (مثل freefire.png):
  - الحجم الموصى به: **500 × 400 بكسل**
  - الصيغة: PNG أو JPG
  - نسبة العرض للارتفاع: 1.25:1

- **Social Icons** (whatsapp.png, facebook.png, etc.):
  - الحجم الموصى به: **200 × 200 بكسل**
  - الصيغة: PNG (خلفية شفافة)

- **Payment Icon** (d17.png):
  - الحجم الموصى به: **150 × 150 بكسل**
  - الصيغة: PNG (خلفية شفافة)

## 🚀 كيفية الاستخدام

### إضافة منتج جديد:

1. ضع صورة المنتج في مجلد `assets/`
2. في ملف `index.html`، جد التعليقات `<!-- PLACEHOLDER: ADD PRODUCT NAME -->` و `<!-- PLACEHOLDER: ADD IMAGE HERE -->`
3. أضف البيانات المطلوبة

### إضافة خبر جديد:

1. ضع صورة الخبر في مجلد `assets/`
2. في ملف `js/script.js`، استخدم الدالة:
```javascript
addNewsArticle('عنوان الخبر', 'assets/image.png', 'محتوى الخبر');
```

### إضافة صور بانر جديدة:

1. ضع صور البانر في مجلد `assets/`
2. في `js/script.js`، أضف اسم الصورة إلى مصفوفة `bannerImages`
